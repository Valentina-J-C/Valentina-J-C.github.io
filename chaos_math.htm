Chaos Theory

The Math Behind It  



Chaotic systems contain one independent variable, usually time, and some dependent variable(s). If one dependent variable is present, it is affected only by the independent variable; however, if multiple dependent variables are present, dependent variables are affected by both the independent variable and other dependent variables.

Recursion

  • The equation(s) affecting dependent variable(s) employ recursion, or the use of a variable’s previous values to determine the current value, via either recursive equations or partial differentials.

  • Iterative equation x(n+1)=f(xn) calculates x(n+1) based on a function f of the previous value xn.

  • Partial derivative ∂x/∂t computes the “velocity” of x as caused by t, while second-order partial derivative (∂2x)/(∂t2) computes the “acceleration” of x as caused by t. These partial derivatives do not consider any effects on velocity or acceleration of x caused by variables other than t; this is achieved by making other variables constant while differentiating, such that they cause no change in x.

Phase Space

  • Continuous chaotic systems are systems in which the independent variable is continuous; that is, its domain includes uncountably infinite values (e.g., all real numbers). Discrete chaotic systems are systems in which the independent variable is quantized; that is, its domain includes only countable numbers (e.g., integers).

  • The dimension of a chaotic system refers to the number of dependent variables describing a given state. The Poincaré-Bendixson Theorem states that continuous chaotic systems must be ≥3-dimensional; however, discrete chaotic systems can be ≥1-dimensional.

  • Because of their multidimensionality, chaotic systems are commonly analyzed within phase space. Within phase space, a range of states over a given independent-variable-range are mapped using Cartesian coordinates against axes corresponding to individual dependent variables; for example, a system containing dependent variables x, y, and z, and independent variable t would map states using these dependent variables for a given t-range within a 3-dimensional phase space using the axes x, y, and z.

  • Because chaotic systems are dynamical, a phase space mapping of all dependent variables for any chaotic system will never cross the same point twice.

Strange Attractor

  • Forced/dampened chaotic systems are systems in which the sum of all dependent variables does not remain constant over a given independent-variable-range; these systems are affected by strange attractors.

  • A strange attractor converges states outside its body towards a trajectory within its body. Paradoxically, they also enact SDIC on states within the attractor, such that two infinitesimally different states within the attractor diverge significantly after a change in the independent variable.

  • Divergence and convergence both occur because the dependent variables all modulate one another; for example, variable x may cause variable y to diverge, which causes variable z to diverge, which causes variable x to converge, and so on, such that an unstable modulation cycle forms a strange attractor.

  • Because chaotic systems are deterministic, the trajectory of a strange attractor never crosses itself; thus, the attractor must be an infinitely complex fractal, giving it fractional dimension. This fractal dimension is what allows the aforementioned modulation cycle to remain unstable.
